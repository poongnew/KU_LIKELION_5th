LEARNING

컴퓨터 개발은 크게 웹서비스의 외관을 구성하는 프론트엔드와 서버 및 데이터를 관리하는 백엔드로 나뉩니다.
멋쟁이 사자처럼의 코딩교육을 통해 프론트엔드와 백엔드에서 사용되는 다양한 언어와 플랫폼을 익힐 수 있습니다.

HTML

HTML은 'HyperText Markup Language'의 약자로서, 웹에 출력되는 콘텐츠를 여러 태그를 이용하여 구성하는 언어입니다. 웹 서비스 구성에서 전체적인 뼈대를 담당합니다.

CSS

CSS은 'Cascading Style Sheet'의 약자로, HTML로 작성된 콘텐츠를 아름답게 꾸미는 역할을 합니다. 하지만 렌더링 되는 원리를 잘 이해해야만 원하는 결과를 얻을 수 있습니다.

JavaScript

JavaScript는 웹 서비스에 동적인 요소를 부가할 때 사용되는 언어이며, 프론트엔드뿐 아니라 백엔드에도 두루 사용됩니다.

Bootstrap

Bootstrap은 twitter에서 개발한 프레임워크로서, 모바일 반응형에 최적화된 프로젝트를 진행할 때 그 빛을 발합니다. 다양한 요소들과 그리드 시스템을 제공하며 오픈 소스입니다.

Ruby on Rails

Ruby on Rails는 Ruby라는 프로그래밍 언어를 사용하여 웹 서비스를 구성하는 프레임워크이며, gem을 통해 손쉽게 기능을 추가할 수 있는 장점이 있습니다.

Git

Git은 프로젝트 버전 관리 시스템으로서 동료들과 협업을 할 때 굉장히 유용하게 사용됩니다. 또한, Github라는 플랫폼을 이용해 웹 상에서 프로젝트를 관리할 수 있습니다.
